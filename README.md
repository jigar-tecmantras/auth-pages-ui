# Auth Pages UI

This repository hosts a pair of polished front-end-only login and registration pages created with Create React App. No backend authentication is included; the forms are purely presentational with client-side validation for demonstration purposes.

## Getting started

```bash
cd frontend
npm install
npm start
```

Then open [http://localhost:3000](http://localhost:3000) to view the application.

## Available scripts

- `npm start` – launch the development server
- `npm run build` – create a production build inside `frontend/build`

## Structure

- `frontend/` – Create React App project
  - `public/index.html` – HTML shell with the root mount point
  - `src/` – React components for Login and Register pages plus shared styles
